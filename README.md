# waleedalvi.com

My personal portfolio — a single-page site built with **SvelteKit 2**, **Svelte 5**, and **TypeScript**. Statically pre-rendered, SEO-ready, and fully responsive.

**[waleedalvi.com](https://waleedalvi.com)**

## Use as a Template

All portfolio content lives in a single file: [`src/lib/data/portfolio.ts`](src/lib/data/portfolio.ts). Replace the data with your own and the site updates accordingly. Styling is controlled through CSS custom properties in [`src/lib/styles/theme.css`](src/lib/styles/theme.css).

```bash
npm install
npm run dev       # local dev server
npm run build     # production build
```

Deploys anywhere SvelteKit supports (Vercel, Netlify, Cloudflare, etc.) via `adapter-auto`.
